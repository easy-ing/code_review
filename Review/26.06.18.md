# **📌 최대공약수와 최대공배수**

**문제 유형**

수학, 유클리드 호제법

---

### 코드

```python
def solution(n, m):
    a, b = n, m

    while b:
        a, b = b, a % b

    gcd = a
    lcm = n * m // gcd

    return [gcd, lcm]
```

---

## **핵심 아이디어**

최대공약수는 유클리드 호제법으로 구한다

최소공배수는 ( a x b ) / 최대공약수

---

## **시간 복잡도**

유클리드 호제법의 시간 복잡도는 O(log(min(n, m)))이다

---

## **총평**

최대공약수와 최소공배수 문제는 사실상 **유클리드 호제법을 알고 있는지 확인하는 문제**이다.

브루트포스로 약수를 전부 찾으면 비효율적이지만,

유클리드 호제법을 사용하면 로그 시간에 해결할 수 있다.
