# **📌 약수의 합**

완전탐색 / 수학 / 구현

---

### 코드

```python
def solution(n):
    answer = 0
    for i in range(1, n+1):
        if n % i == 0:
            answer += i
    return answer
```

---

### **⚠️ 아쉬운 점**

**1. 불필요하게 n까지 모두 탐색한다**
약수는 항상 쌍으로 존재합니다.

예:

12 → (1,12), (2,6), (3,4)

즉,

√n까지만 탐색해도 모든 약수를 찾을 수 있습니다.

---

### **개선 포인트 설명**

- `i`는 √n까지만 확인
- 약수 `i`를 찾으면 → `n // i`도 함께 더함
- 단, `i == n//i` (완전제곱수)일 때는 한 번만 더함

---

### **❌ 자주 틀리는 포인트**

- `range(n)` → n 포함 안 됨 ❌
- `i == n//i` 처리 안 해서 중복 더함 ❌
- 0 입력 시 예외 고려 부족 ❌

---

### **✅ 총평**

현재 풀이는

**가장 정석적인 완전탐색 풀이로, 정확성과 안정성이 매우 좋은 코드**입니다.

다만 약수의 성질을 활용하면

**탐색 범위를 절반 이하로 줄일 수 있는 개선 여지**가 있습니다.

즉,

- 현재 코드 → 직관적이고 안전한 기본 풀이
- 개선 코드 → 수학적 최적화를 적용한 효율적인 풀이
