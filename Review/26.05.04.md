# **📌 하샤드 수**

### **문제 유형**

문자열 / 수학 / 구현

---

### 코드

```python
def solution(x):
    digit_sum = sum(int(c) for c in str(x))
    return x % digit_sum == 0
```

---

### **❌ 자주 틀리는 포인트**

- 자릿수 합을 문자열 변환 없이 처리하다 실수 ❌
- 나눗셈(`/`) 사용 → 나머지 조건이 아니라 몫 계산 ❌
- 조건문 길게 쓰는 경우 (불필요) ❌

---

### **✅ 총평**

이 문제는

**자릿수 처리 + 나머지 조건**을 묻는 기본 구현 문제입니다.

핵심은:

- 자릿수 합 구하기
- 나누어 떨어지는지 확인

즉,

**문제 정의를 그대로 구현하면 되는 깔끔한 문제**입니다.
