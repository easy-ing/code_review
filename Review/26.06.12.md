# **📌 다트 게임**

**문제 유형**

문자열 파싱, 구현, 스택/리스트 활용

---

### 코드

```python
def solution(dartResult):
    scores = []
    i = 0

    while i < len(dartResult):
        if dartResult[i].isdigit():
            if i + 1 < len(dartResult) and dartResult[i + 1].isdigit():
                score = int(dartResult[i:i+2])
                i += 2
            else:
                score = int(dartResult[i])
                i += 1

            bonus = dartResult[i]

            if bonus == "S":
                score = score ** 1
            elif bonus == "D":
                score = score ** 2
            elif bonus == "T":
                score = score ** 3

            scores.append(score)
            i += 1

        else:
            if dartResult[i] == "*":
                scores[-1] *= 2

                if len(scores) >= 2:
                    scores[-2] *= 2

            elif dartResult[i] == "#":
                scores[-1] *= -1

            i += 1

    return sum(scores)
```

---

## **핵심 아이디어**

다트 게임은 총 3번의 점수를 계산해야 하므로, 각 점수를 계산해서 리스트에 저장하면 된다.

문자열을 왼쪽부터 읽으면서

- 숫자면 점수로 저장
- `S`, `D`, `T`면 제곱 적용
- 면 현재 점수와 이전 점수 2배
- `#`면 현재 점수 음수 처리

를 하면 된다.

주의할 점은 점수가 `10`일 수 있다는 것이다.

그래서 숫자를 읽을 때 한 자리 숫자만 보면 안 되고, `10`인지 확인해야 한다.

---

## **시간 복잡도**

문자열 `dartResult`를 왼쪽부터 한 번만 순회한다.

따라서 시간복잡도는 O(N)이다

---

## **총평**

이 문제는 계산 자체보다 **문자열을 정확히 파싱하는 것**이 핵심이다.

특히 주의할 부분은 세 가지다.

- 점수 `10` 처리
- 가 나오면 현재 점수와 이전 점수를 함께 2배 처리
- `#`가 나오면 현재 점수만 음수 처리

각 라운드 점수를 리스트에 저장하면 `*` 옵션이 이전 점수에도 영향을 주는 처리를 쉽게 할 수 있다.
