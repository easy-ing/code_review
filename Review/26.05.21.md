# **📌 핸드폰 번호 가리기**

**문제 유형**

문자열(String), 슬라이싱(Slicing)

---

### 코드

```python
def solution(phone_number):
    answer = '*' * (len(phone_number) - 4) + phone_number[-4:]
    return answer
```

---

## **핵심 아이디어**

- 전화번호의 마지막 4자리는 그대로 유지
- 앞부분은  문자로 변환
- `len(phone_number) - 4` 만큼  생성
- 마지막 4자리는 슬라이싱 `[-4:]` 사용

---

## **시간 복잡도**

- 문자열 길이를 `n`이라고 할 때
- 생성: O(n)
- 슬라이싱: O(4)

따라서 전체 시간 복잡도는: O(n)

---

## **총평**

문자열 슬라이싱과 문자열 곱셈을 활용하여 매우 간결하게 구현한 코드입니다.

특히 `phone_number[-4:]`를 사용해 마지막 4자리를 쉽게 추출한 점이 깔끔하며, Python 문자열 처리의 장점을 잘 활용한 풀이입니다.
