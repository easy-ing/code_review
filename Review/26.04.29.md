# **📌 문자열을 정수로 바꾸기**

문자열 / 형변환 / 구현

---

### 코드

```python
def solution(s):
    return int(s)
```

---

### **💡 핵심 아이디어**

이 문제는 직접 파싱할 필요 없이

파이썬 내장 함수인 int()를 사용하면 됩니다.

- "1234" → 1234
- "-1234" → -1234
- "+1234" → 1234

부호 처리까지 자동으로 해주기 때문에

별도 로직이 필요 없습니다.

---

### **👍 잘한 점**

**1. 내장 함수를 활용한 최적의 풀이**

문제 조건이 이미 안전하게 주어져 있기 때문에

불필요한 구현 없이 int() 한 줄로 해결 가능합니다.

---

**2. 부호 처리까지 자동 해결**

직접 +, -를 나눠서 처리할 필요 없이

내장 함수가 모두 처리해줍니다.

---

**3. 가독성과 유지보수성이 뛰어나다**

불필요한 코드 없이 의도가 명확합니다.

---

### **❌ 자주 틀리는 포인트**

- 문자열을 직접 숫자로 변환하려고 반복문 사용 ❌
- 부호 처리 따로 구현 ❌
- int() 사용을 놓침 ❌

---

### **✅ 총평**

이 문제는 알고리즘 문제가 아니라

**기본적인 언어 활용 능력**을 묻는 문제입니다.

핵심은 하나입니다.

- int()를 알고 있는가?

즉,

**불필요한 구현보다 내장 기능을 활용하는 것이 가장 좋은 풀이**입니다.
