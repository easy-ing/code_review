# **📌 문자열 다루기 기본**

**문제 유형**

문자열, 조건문, 문자열 함수 적용

---

### 코드

```python
def solution(s):
    return (len(s) == 4 or len(s) == 6) and s.isdigit()
```

---

## **핵심 아이디어**

- 문자열 길이가:
    - 4 또는 6인지 확인
- 문자열이:
    - 숫자로만 이루어져 있는지 확인
- `isdigit()` 함수 사용:
    - 문자열이 모두 숫자이면 `True`
    - 하나라도 문자가 포함되면 `False`

---

## **시간 복잡도**

- 문자열 길이를 `n`이라고 할 때
- `isdigit()`은 문자열 전체를 확인하므로 O(n)

따라서 전체 시간 복잡도는:

O(n)

---

## **총평**

Python의 내장 함수 `isdigit()`을 활용하여 매우 간결하게 구현한 코드입니다.

길이 조건과 숫자 여부를 논리 연산자로 한 번에 처리하여 가독성이 좋으며, 불필요한 반복문 없이 효율적으로 문제를 해결한 풀이입니다.
