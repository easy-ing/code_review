# **📌 문자열 치환 로직 코드 리뷰**

## **🧾 문제 요약**

입력 문자열 myString이 주어졌을 때,

각 문자를 순회하면서 **‘l’보다 작은 문자는 모두 ‘l’로 치환**하고,

그 외 문자는 그대로 유지한 새로운 문자열을 반환한다.

---

## **💻 코드**

```java
class Solution {
    public String solution(String myString) {
        StringBuilder sb = new StringBuilder(myString.length());

        for (int i = 0; i < myString.length(); i++) {
            char ch = myString.charAt(i);
            if (ch < 'l') sb.append('l');
            else sb.append(ch);
        }

        return sb.toString();
    }
}
```

---

## **🔍 동작 방식 (Step-by-Step)**

1. StringBuilder 생성
    - 문자열을 효율적으로 생성하기 위한 객체
    - 초기 용량을 myString.length()로 설정 → 불필요한 메모리 재할당 방지
2. 문자열 순회 (for)
    - 인덱스를 기반으로 한 문자 접근
3. 문자 비교 (ch < 'l')
    - 아스키/유니코드 값 기준 비교
    - 'a' ~ 'k' → 'l'로 치환
4. 결과 문자열 생성
    - 조건에 따라 'l' 또는 원래 문자 append
5. 최종 문자열 반환

## **✨ 리팩토링 코드 (실무 스타일)**

```java
class Solution {
    public String normalizeString(String input) {
        final char 기준문자 = 'l';
        StringBuilder result = new StringBuilder(input.length());

        for (int i = 0; i < input.length(); i++) {
            char currentChar = input.charAt(i);
            result.append(currentChar < 기준문자 ? 기준문자 : currentChar);
        }

        return result.toString();
    }
}
```
