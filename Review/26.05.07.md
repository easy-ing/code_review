# **📌정수 제곱근 판별**

### **문제 유형**

수학 / 구현 / 제곱근

---

### 코드

```python
def solution(n):
    answer = 0

    for i in range(0, n + 1, 1):
        if n == i * i:
            answer = (i + 1) * (i + 1)
        else:
            answer = -1

    return answer
```

---

### **❌ 자주 틀리는 포인트**

- `(a)(b)` 형태로 곱셈 작성 ❌
- else를 반복문 안에 둠 ❌
- 완전탐색으로 50조까지 순회 ❌
- `n ** 0.5 == int(n ** 0.5)`처럼 float 비교 ❌
