## 문제

수 N개가 주어졌을 때, i번째 수부터 j번째 수까지 합을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 수의 개수 N과 합을 구해야 하는 횟수 M이 주어진다. 둘째 줄에는 N개의 수가 주어진다. 수는 1,000보다 작거나 같은 자연수이다. 셋째 줄부터 M개의 줄에는 합을 구해야 하는 구간 i와 j가 주어진다.

## 출력

총 M개의 줄에 입력으로 주어진 i번째 수부터 j번째 수까지 합을 출력한다.

## 제한

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 1 ≤ i ≤ j ≤ N

## 예제 입력 1

```
53
54321
13
24
55
```

## 예제 출력 1

```
12
9
1
```

---

## 내가 작성한 코드

```java
import sys

input = sys.stdin.readline

N, M = map(int, input().split())
numbers = list(map(int, input().split()))

# 누적합 배열
prefix = [0] * (N + 1)
for i in range(1, N + 1):
    prefix[i] = prefix[i - 1] + numbers[i - 1]

# 구간합 출력
for _ in range(M):
    i, j = map(int, input().split())
    print(prefix[j] - prefix[i - 1])
```

### **총평**

- **좋은 점**
    - 시간복잡도를 잘 고려해서 누적합(prefix sum)으로 풀이함
    - 입력 크기 N, M <= 100000을 보고 sys.stdin.readline을 사용한 점이 좋음
    - 구현이 짧고, 문제 의도에 정확히 맞음
    - 불필요한 연산이 없음
- **아쉬운 점**
    - 변수명이 조금 더 의도가 드러나면 좋음
    - 실무 코드 기준으로는 “왜 이렇게 하는지”를 설명하는 주석이 조금 더 있으면 가독성이 좋아짐
    - 출력도 한 줄씩 print() 하는 대신 모아서 출력하면 더 일관되게 성능을 챙길 수 있음
