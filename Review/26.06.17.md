# **📌 전력망을 둘로 나누기**

**문제 유형**

그래프, 트리, BFS/DFS탐색, 브루트포스

---

### 코드

```python
from collections import deque

def solution(n, wires):
    answer = n

    for cut in range(len(wires)):
        graph = [[] for _ in range(n + 1)]

        # cut번째 전선만 빼고 그래프 만들기
        for i, (a, b) in enumerate(wires):
            if i == cut:
                continue
            graph[a].append(b)
            graph[b].append(a)

        # 1번 송전탑 쪽 네트워크 개수 세기
        visited = [False] * (n + 1)
        queue = deque([1])
        visited[1] = True
        count = 1

        while queue:
            now = queue.popleft()

            for next_node in graph[now]:
                if not visited[next_node]:
                    visited[next_node] = True
                    queue.append(next_node)
                    count += 1

        other = n - count
        answer = min(answer, abs(count - other))

    return answer
```

---

## **핵심 아이디어**

트리는 간선이 `n-1개`인 연결 그래프이다.

전선을 하나 끊으면 반드시 그래프가 **두 개의 트리**로 분리된다.

따라서 모든 전선에 대해 다음 과정을 반복하면 된다.

1. 전선 하나를 제거한다.
2. BFS(또는 DFS)로 한 쪽 전력망의 송전탑 개수를 센다.
3. 나머지 전력망의 개수는 `n - count` 이다.
4. 두 전력망의 차이
5. 가장 작은 차이를 정답으로 갱신한다.

즉,

“모든 전선을 하나씩 끊어보고, 나뉘는 두 전력망의 크기 차이 중 최솟값을 찾는다.”

가 핵심이다.

---

## **시간 복잡도**

전선의 개수는 `n - 1`개

각 전선을 끊을 때마다

- 그래프 생성 : O(n)
- BFS 탐색 : O(n)

이 수행된다.

따라서 전체 시간 복잡도는 

O(n × n)
= O(n²)

## **총평**

이 문제의 핵심은

“간선을 하나 제거하면 트리가 정확히 두 개의 연결 요소로 나뉜다”

는 성질을 이용하는 것이다.

프로그래머스에서 BFS/DFS 입문 단계에서 자주 나오는 유형으로,

- 그래프 생성
- 방문 배열 사용
- 연결 요소 크기 구하기

연습에 좋은 문제이다.
