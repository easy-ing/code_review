# **📌 대충 만든 자판**

**문제 유형**

그리디, 자리수 계산, 수학, 구현

---

### 코드

```python
def solution(storey):
    answer = 0

    while storey > 0:
        digit = storey % 10
        next_digit = (storey // 10) % 10

        if digit < 5:
            answer += digit
            storey //= 10

        elif digit > 5:
            answer += 10 - digit
            storey = storey // 10 + 1

        else:
            if next_digit >= 5:
                answer += 5
                storey = storey // 10 + 1
            else:
                answer += 5
                storey //= 10

    return answer
```

---

## **핵심 아이디어**

각 자리수를 보면서 해당 자리 숫자를

- 내려서 `0`으로 만들지
- 올려서 다음 자리로 넘길지

를 결정하는 문제다.

예를 들어 `16`에서 1의 자리 `6`은

- `1`을 6번 누르기
- `+1`을 4번 눌러서 20으로 만들고, 10의 자리에서 처리하기

중 선택할 수 있다.

`6`은 5보다 크기 때문에 올리는 쪽이 더 적다.

---

## **시간 복잡도**

`storey`를 10으로 나누면서 각 자리수를 한 번씩 확인한다.

`storey`의 최대값은 `100,000,000`이므로 자리수는 최대 9자리다.

따라서 시간복잡도는 O(log N)이다.

---

## **총평**

이 문제는 단순히 각 자리 숫자가 `5`보다 크면 올리고, 작으면 내리는 방식으로 풀 수 있다.

다만 숫자가 정확히 `5`일 때가 핵심이다.

현재 자리가 `5`라면 다음 자리 숫자를 확인해서

- 다음 자리 숫자가 5 이상이면 올림
- 다음 자리 숫자가 5 미만이면 내림

으로 처리해야 최소 횟수를 구할 수 있다.
