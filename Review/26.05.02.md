# **📌 x만큼 간격이 있는 n개의 숫자**

구현 / 리스트 생성 / 반복문

---

### 코드

```python
def solution(x, n):
    return [x * i for i in range(1, n + 1)
```

---

### **❌ 자주 틀리는 포인트**

- range(n) 사용 후 i 그대로 곱하기 → 0 포함됨 ❌
- range(0, n) → 첫 값이 0이 되어버림 ❌
- n개가 아니라 n+1개 생성 ❌

---

### **✅ 총평**

이 문제는 단순 구현 문제지만

**패턴을 일반화해서 표현할 수 있는지**를 보는 문제입니다.

핵심은:

- x * i 형태로 생각하기
- i를 1부터 n까지 돌리기

즉,

**기본적인 반복과 리스트 생성 능력을 묻는 문제**입니다.
