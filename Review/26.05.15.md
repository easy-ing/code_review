# **📌 가운데 글자 가져오기**

**문제 유형**

문자열(String) 인덱싱 및 슬라이싱

---

### 코드

```python
def solution(s):
    mid = len(s) // 2

    if len(s) % 2 == 0:
        return s[mid - 1:mid + 1]

    else:
        return s[mid]
```

---

## **핵심 아이디어**

- 문자열 길이가 홀수라면:
    - 가운데 문자 1개 반환
- 문자열 길이가 짝수라면:
    - 가운데 문자 2개 반환

---

## **시간 복잡도**

- 문자열 길이 계산: O(1)
- 슬라이싱: 최대 2글자

따라서 전체 시간 복잡도는: O(1)

---

## **총평**

문자열 인덱싱과 슬라이싱을 활용하여 문제를 직관적으로 해결한 깔끔한 코드입니다.

특히 짝수/홀수 조건을 명확하게 나눈 점이 좋으며, Python 문자열 처리의 기본기를 잘 활용한 풀이입니다.
