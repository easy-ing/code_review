# **📌 리프 노드 수 최대화**

**문제 유형**

문자열, 조건문

---

### 코드

```python
code = input()
last_four_words = code[-4:]

if last_four_words == "_eye":
    print("Ophthalmologyc")
elif last_four_words == "head":
    print("Neurosurgery")
elif last_four_words == "infl":
    print("Orthopedics")
elif last_four_words == "skin":
    print("Dermatology")
else:
    print("direct recommendation")
```

---

## **핵심 아이디어**

- 환자 코드의 마지막 4글자를 추출
- 마지막 4글자에 따라 병과를 구분
- 조건문을 사용하여 각 병과명을 출력
- 지정된 코드가 아니라면 `"direct recommendation"` 출력

---

## **시간 복잡도**

- 마지막 4글자 추출: O(1)
- 조건 비교: 최대 4회

따라서 전체 시간 복잡도는:

**O(1)**

---

## **총평**

문자열 슬라이싱을 이용하여 환자 코드의 마지막 4글자를 간단하게 추출한 뒤, 조건문으로 병과를 분류한 직관적인 풀이입니다.

특히 `code[-4:]`를 활용하여 필요한 정보만 추출한 점이 깔끔하며, `if-elif-else` 구조를 사용해 병과 분류 기준을 명확하게 표현했습니다.

문자열 처리와 조건문 활용 능력을 확인할 수 있는 기본적인 구현 문제로, 가독성이 좋은 코드입니다.
