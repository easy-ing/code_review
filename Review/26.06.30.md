# **📌 숫자 문자열과 영단어**

**문제 유형**

문자열, 딕셔너리, 치환

---

### 코드

```python
def solution(s):
    numbers = {
        "zero": "0",
        "one": "1",
        "two": "2",
        "three": "3",
        "four": "4",
        "five": "5",
        "six": "6",
        "seven": "7",
        "eight": "8",
        "nine": "9"
    }

    for word, num in numbers.items():
        s = s.replace(word, num)

    return int(s)
```

---

## **핵심 아이디어**

영단어로 적힌 숫자를 실제 숫자로 바꾸면 된다.

파이썬에서 replace()를 사용하면 쉽게 해결할 수 있다.

---

## **시간 복잡도**

영단어는 총 10개이고, 문자열 길이는 최대 50이다

각 단어마다 replace를 수행하니까 O(10 X N)이다

즉 O(N)이다.

---

## **총평**

이 문제는 복잡한 파싱보단 문자열 치환으로 푸는게 가장 간단하다.

핵심은 숫자 영단어와 실제 숫자를 딕셔너리를 매핑한 뒤,

s.replace(word, num)로 전부 바꾸는 것이다.
