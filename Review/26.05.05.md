# **📌평균 구하기**

### **문제 유형**

배열 / 수학 / 구현

---

### 코드

```python
def solution(arr):
    return sum(arr) / len(arr)
```

---

### **❌ 자주 틀리는 포인트**

- `//` 사용 → 정수 나눗셈이라 소수점 날아감 ❌
- `len(arr)` 빼먹음 ❌
- 반복문으로 합 구하다 실수 ❌

---

### **✅ 총평**

이 문제는

**기본적인 배열 처리 + 수학 공식 적용 문제**입니다.

핵심은:

- 총합 구하기
- 개수로 나누기

즉,

**내장 함수 활용 능력을 묻는 가장 기초적인 문제**입니다.
