# **📌 시소 짝궁**

**문제 유형**

해시맵, 빈도계산, 수학, 정렬

---

### 코드

```python
from collections import defaultdict

def solution(weights):
    answer = 0
    count = defaultdict(int)

    weights.sort()

    for w in weights:
        answer += count[w]

        if w * 2 % 3 == 0:
            answer += count[w * 2 // 3]

        if w % 2 == 0:
            answer += count[w // 2]

        if w * 3 % 4 == 0:
            answer += count[w * 3 // 4]

        count[w] += 1

    return answer
```

---

## **핵심 아이디어**

시소가 균형을 이루려면

몸무게 X 거리 = 몸무게 X 거리가 같아야한다.

좌석거리는 2m, 4m이렇게 두개니까 비율은

1:1  / 2: 3 / 1 : 2 / 3:4 이렇게 가능하다.

따라서 몸무게 오름차순 정렬한 뒤, 현재 몸무게 w와 짝이 될수있는 이전 몸무게만 보면된다.

---

## **시간 복잡도**

몸무게 정렬한다 → 정렬 시간 복잡도 O(N log N)

이후 weight 한번 순회한다. 근데 이거 비율은 고정이니까 O(N)

따라서 전체 시간 복잡도는 O(N log N)

---

## **총평**

초과가 난다.

그래서 이미 등장한 몸무게의 개수를 딕셔너리에 저장해두고, 현재 몸무게와 짝이 될 수 있는 몸무게가 이전에 있었는지만 확인해야 한다.

핵심은 **정렬 후 현재 몸무게보다 작거나 같은 후보만 찾는 것**이다.

가능한 비율이 제한되어 있기 때문에 `1:1`, `2:3`, `1:2`, `3:4` 네 가지 경우만 확인하면 된다.
