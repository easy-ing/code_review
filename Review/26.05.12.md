# **📌 문자열 내 p와 y의 개수**

### **문제 유형**

수학 / 구현 / 조건문

---

### 코드

```python
def solution(s):
    s = s.lower()

    return s.count('p') == s.count('y')
```

---

## **총평**

Python의 문자열 메서드인 `lower()`와 `count()`를 적절히 활용하여 매우 간결하고 직관적으로 해결한 코드입니다.

특히 조건식을 바로 반환한 점이 깔끔하며, 코딩테스트에서 자주 사용되는 Pythonic한 풀이 방식에 가깝습니다.

---

## **시간 복잡도**

### **현재 코드**

- `lower()` : O(n)
- `count('p')` : O(n)
- `count('y')` : O(n)

총 시간 복잡도:

O(n)

문자열 길이가 최대 50이므로 충분히 효율적입니다.

---

## **핵심 아이디어**

- 문자열을 모두 소문자(또는 대문자)로 통일하면 대소문자 구분 없이 비교할 수 있습니다.
- `count()` 함수를 사용하면 특정 문자의 개수를 쉽게 셀 수 있습니다.
- `'p'`와 `'y'`의 개수가 같으면 `True`, 다르면 `False`를 반환합니다.
