# **📌 문자열 내 마음대로 정렬하기**

**문제 유형**

문자열, 정렬, 람다 함수

---

### 코드

```python
def solution(strings, n):
    return sorted(strings, key=lambda x: (x[n], x))
```

---

## **핵심 아이디어**

정렬 기준이 2개이다.

1. n번째 문자 기준 오름차순
2. n번째 문자가 같으면 문자열 자체를 사전순 정렬

---

## **시간 복잡도**

문자열 개수를 n이라고 하면 정렬이 수행되므로 O(N log N)이다.

---

## **총평**

이 문제의 핵심은 **정렬 기준을 2개 설정하는 것**이다.

파이썬의 `sorted()`는 튜플을 기준으로 정렬할 수 있으므로

key=lambda x: (x[n], x)만으로

- n번째 문자 정렬
- 문자열 사전순 정렬을 동시에 처리할 수 있다.

정렬 문제 입문에서 자주 나오는 **다중 조건 정렬** 유형이다.
