# **📌 Collatz 추측**

**문제 유형**

반복문(While), 조건문(If), 구현(Implementation)

---

### 코드

```python
def solution(num):
    count = 0
    
    while num != 1:
        if num % 2 == 0:
            num = num // 2
        else:
            num = num * 3 + 1
        
        count += 1
        
        if count == 500:
            return -1
    
    return count
```

---

## **핵심 아이디어**

- 숫자가 1이 될 때까지 반복 수행
- 짝수라면:
    - 2로 나누기
- 홀수라면:
    - 3을 곱하고 1 더하기
- 반복 횟수를 `count`로 저장
- 500번 반복해도 1이 되지 않으면 `1` 반환

---

## **시간 복잡도**

- 최악의 경우 최대 500번 반복

따라서 전체 시간 복잡도는: O(1)

(문제에서 반복 횟수가 500 이하로 제한되어 있기 때문)

---

## **총평**

반복문과 조건문을 활용하여 Collatz 추측 과정을 직관적으로 구현한 코드입니다.

짝수와 홀수 조건을 명확하게 분리하였고, 반복 횟수 제한 조건까지 깔끔하게 처리한 점이 좋습니다.

또한 `//` 연산자를 사용해 정수 형태를 유지한 점도 안정적인 구현 방식입니다.
