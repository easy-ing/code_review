# **📌 행렬의 덧셈**

**문제 유형**

2차원 배열(Array), 반복문(For), 구현(Implementation

---

### 코드

```python
def solution(arr1, arr2):
    answer = []
    
    for i in range(len(arr1)):
        row = []
        
        for j in range(len(arr1[i])):
            row.append(arr1[i][j] + arr2[i][j])
        
        answer.append(row)
    
    return answer
```

---

## **핵심 아이디어**

- 같은 위치의 원소끼리 더해야 함
- 바깥 반복문:
    - 행(row) 순회
- 안쪽 반복문:
    - 열(column) 순회
- `arr1[i][j] + arr2[i][j]` 값을 새로운 행렬에 저장

---

## **시간 복잡도**

- 행 개수를 `n`
- 열 개수를 `m`이라고 할 때

모든 원소를 한 번씩 확인하므로:

O(n × m)

---

## **총평**

이중 반복문을 활용하여 행과 열을 순차적으로 탐색하며 행렬 덧셈을 정확하게 구현한 코드입니다.

특히 결과 행(row)을 따로 생성한 뒤 `answer`에 추가하는 방식으로 2차원 배열 구조를 깔끔하게 구성한 점이 좋습니다.
