# **📌 삼총사**

**문제 유형**

완전탐색, 조합, 브루트포스

---

### 코드

```python
def solution(number):
    answer = 0
    n = len(number)

    for i in range(n):
        for j in range(i + 1, n):
            for k in range(j + 1, n):
                if number[i] + number[j] + number[k] == 0:
                    answer += 1

    return answer
```

---

## **핵심 아이디어**

삼총사는 서로 다른 학생 3명의 번호 합이 0인경우다

따라서 가능한 모든 3명의 조합을 확인하면 된다.

number[i] + number[j] + number[k] == 0 이면 정답을 1 증가시킨다

---

## **시간 복잡도**

학생 수를 N이라고 하면 3명을 뽑는 모든 경우를 확인하니

시간 복잡도는 O(N³)이다

---

## **총평**

이 문제는 효율적인 알고리즘보다 **정확하게 모든 조합을 확인하는 것**이 중요하다.

특히 같은 번호가 있어도 학생이 다르면 다른 경우로 봐야 하므로, 값 기준이 아니라 **인덱스 기준**으로 3명을 뽑아야 한다.
