# **📌 부족한 금액 계산하기**

**문제 유형**

반복문, 누적합, 구현

---

### 코드

```python
def solution(price, money, count):
    total = 0
    
    for i in range(1, count + 1):
        total += price * i
    
    if total <= money:
        return 0
    
    return total - money
```

---

## **핵심 아이디어**

- 놀이기구를 탈 때마다 이용료가 증가
    - 1번째: `price × 1`
    - 2번째: `price × 2`
    - …
    - N번째: `price × N`
- 반복문을 사용하여 총 이용 금액 계산
- 현재 가진 돈 `money`와 비교
    - 부족하지 않으면 `0`
    - 부족하면 `총 금액 - money` 반환

---

## **시간 복잡도**

- `count`번 반복 수행

따라서 전체 시간 복잡도는:

O(count)

---

## **총평**

반복문을 사용해 이용 횟수마다 증가하는 금액을 누적 합으로 계산한 직관적인 풀이입니다.

조건문을 통해 부족한 경우와 부족하지 않은 경우를 명확하게 처리하였으며, 변수명 또한 의미가 분명해 가독성이 좋은 코드입니다.

구현 문제의 기본 흐름을 깔끔하게 잘 담아낸 풀이입니다.
