# **📌이진변환 반복하기**

### **문제 유형**

문자열 / 구현 / 반복문

---

### 코드

```python
def solution(s):
    transform_count = 0
    removed_zero_count = 0

    while s != "1":
        zero_count = s.count('0')
        removed_zero_count += zero_count

        one_count = len(s) - zero_count
        s = bin(one_count)[2:]

        transform_count += 1

    return [transform_count, removed_zero_count]**💡 핵심 아이디어**
```

---

### **💡 핵심 아이디어**

이 문제의 핵심은 매 단계에서 해야 할 작업이 항상 똑같다는 점입니다.

1. 문자열에서 0의 개수를 센다.
2. 그만큼 제거된 0의 개수를 누적한다.
3. 남은 1의 개수만큼 길이를 구한다.
4. 그 길이를 다시 2진수 문자열로 바꾼다.
5. 문자열이 "1"이 될 때까지 반복한다.

즉, 문자열 자체를 직접 필터링해서 "111..."를 만들 필요 없이

**0의 개수와 1의 개수만 계산하면 충분한 문제**입니다.

---

### **👍 잘한 점**

**1. 불필요한 문자열 재구성을 줄일 수 있다**

0을 하나씩 지우는 방식보다

count('0')으로 0의 개수를 세고,

전체 길이에서 빼서 1의 개수를 구하면 훨씬 간단합니다.

**2. 문제 조건을 그대로 코드로 옮기기 쉽다**

문제에서 요구하는 과정이 명확해서

while s != "1" 반복문 안에 순서대로 구현하면 됩니다.

**3. 시간 복잡도도 충분히 안정적이다**

매 반복마다 문자열 길이가 줄어들기 때문에

전체적으로 매우 효율적으로 동작합니다.

---

### **⚠️ 아쉬운 점**

불필요하게 복잡한 코드를 작성하는 것입니다.
