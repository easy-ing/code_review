# **📌 정수 내림차순으로 배치하기**

### **문제 유형**

수학 / 구현 / 반복문

---

### 코드

```python
def solution(n):
    digits = list(str(n))
    digits.sort(reverse=True)
    return int(''.join(digits))
```

---

## **핵심 아이디어**

- 숫자를 문자열로 변환하면 각 자릿수를 쉽게 분리할 수 있습니다.
- `sort(reverse=True)`를 사용하면 큰 숫자부터 정렬할 수 있습니다.
- 정렬된 문자열을 다시 이어 붙인 뒤 `int()`로 변환하여 반환합니다.

---

## **시간 복잡도**

- 문자열 길이를 k라고 할 때
- 정렬 수행 비용:

O(k \log k)

최대 숫자 범위가 크지 않아 충분히 효율적인 풀이입니다.

---

## **총평**

문자열 변환과 정렬을 적절히 활용하여 문제를 매우 간결하고 안정적으로 해결한 코드입니다.

불필요한 반복문이나 복잡한 수학 연산 없이 Python의 내장 기능을 잘 활용한 풀이이며, 실무에서도 선호되는 깔끔한 스타일에 가깝습니다.
