# **📌 직사각형 별찍기**

**문제 유형**

문자열, 반복문

---

### 코드

```python
a, b = map(int, input().strip().split(' '))

for i in range(b):
    print('*' * a)
```

---

## **핵심 아이디어**

- 가로 길이 `a`
    - `'*' * a` 를 사용하여 한 줄 생성
- 세로 길이 `b`
    - 반복문을 사용하여 `b`번 출력

예를 들어:

- `a = 5`
- `b = 3`

이라면 `"*****"`를 3번 출력

---

## **시간 복잡도**

- 세로 `b`번 반복
- 각 줄마다 길이 `a` 문자열 생성

따라서 전체 시간 복잡도는:

O(a × b)

---

## **총평**

문자열 곱셈과 반복문을 활용하여 직사각형 형태를 직관적으로 출력한 코드입니다.

특히 `'*' * a`를 사용해 한 줄을 간단하게 생성한 점이 Python다운 깔끔한 구현이며, 문제의 요구사항을 명확하게 반영한 풀이입니다.
