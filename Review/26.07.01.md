# **📌 명예의전당(1)**

**문제 유형**

정렬, 구현, 시물레이션

---

### 코드

```python
def solution(k, score):
    answer = []
    hall = []

    for s in score:
        hall.append(s)
        hall.sort(reverse=True)

        if len(hall) > k:
            hall.pop()

        answer.append(hall[-1])

    return answer
```

---

## **핵심 아이디어**

매일 점수를 하나씩 확인하면서 **명예의 전당 리스트**를 관리하면 된다.

규칙은 간단하다.

1. 오늘 점수를 명예의 전당 후보에 넣는다.
2. 점수를 내림차순으로 정렬한다.
3. 상위 `k개`만 남긴다.
4. 그중 가장 낮은 점수를 `answer`에 넣는다.

---

## **시간 복잡도**

score의 길이를 N이라고 하면, 매번 정렬을 하므로

O(N X k log K)이다

---

## **총평**

이 문제는 매일 명예의 전당을 갱신하는 **시뮬레이션 문제**이다.

핵심은 명예의 전당에는 최대 `k개`까지만 유지하고, 매일 그중 최하위 점수인 `hall[-1]`을 기록하는 것이다.
