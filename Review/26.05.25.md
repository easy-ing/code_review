# **📌 문자열 내림차순으로 배치하기**

**문제 유형**

문자열, 정렬

---

### 코드

```python
def solution(s):
    answer = ''.join(sorted(s, reverse=True))
    return answer
```

---

## **핵심 아이디어**

- 문자열의 각 문자를 정렬
- `sorted()` 함수 사용:
    - 문자열을 문자 단위로 정렬한 리스트 반환
- `reverse=True` 옵션:
    - 내림차순 정렬 수행
- `join()` 사용:
    - 정렬된 문자 리스트를 다시 문자열로 변환

---

## **시간 복잡도**

- 문자열 길이를 `n`이라고 할 때
- 정렬 수행: O(n log n)

따라서 전체 시간 복잡도는:

O(n log n)

---

## **총평**

Python의 `sorted()`와 `join()` 함수를 활용하여 매우 간결하게 구현한 코드입니다.

특히 `reverse=True` 옵션을 통해 내림차순 정렬을 직관적으로 표현한 점이 좋으며, 문자열 정렬 문제를 Python스럽게 해결한 깔끔한 풀이입니다.
