# **📌두 정수 사이의 합**

수학 / 구현 / 등차수열

---

### 코드

```python
def solution(a, b):
    start = min(a, b)
    end = max(a, b)

    count = end - start + 1
    return (start + end) * count // 2
```

---

### **💡 핵심 아이디어**

이 문제는 단순 반복으로 풀 수도 있지만,

**등차수열 합 공식**을 쓰면 더 깔끔하게 해결됩니다.

등차수열 합:

(첫항 + 마지막항) \times 개수 \div 2

즉,

- 시작값: min(a, b)
- 끝값: max(a, b)
- 개수: (end - start + 1)

을 이용하면 반복문 없이 한 번에 계산할 수 있습니다.

---

### **👍 잘한 점**

**1. 반복문 없이 O(1)로 해결 가능하다**

범위가 최대 2천만이라 반복문으로도 가능하지만,

공식을 쓰면 훨씬 효율적입니다

**2. a, b의 대소 관계를 미리 정리했다**

**3. 정수 나눗셈 처리까지 정확하다**

결과를 정수로 유지한 점도 정확합니다

### **✅ 총평**

이 문제는 구현 문제처럼 보이지만,

사실은 **등차수열 공식 적용 문제**입니다.

핵심은 다음입니다.

- 시작값 / 끝값 정리
- 항의 개수 계산
- 공식 적용

즉,

**수학적 접근을 하면 훨씬 간결하게 풀리는 문제**입니다.
