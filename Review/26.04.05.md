# **📌 2차원 누적합 코드리뷰**

## **🧾 문제 유형**

- 2차원 구간 합 (Range Sum Query)
- 핵심: **2차원 누적합 (Prefix Sum)**

---

# **제출 코드**

```python
import sys
input = sys.stdin.readline

N, M = map(int, input().split())
prefix = [[0] * (N + 1) for _ in range(N + 1)]

for i in range(1, N + 1):
    row = list(map(int, input().split()))
    for j in range(1, N + 1):
        prefix[i][j] = prefix[i - 1][j] + prefix[i][j - 1] - prefix[i - 1][j - 1] + row[j - 1]

answers = []
for _ in range(M):
    x1, y1, x2, y2 = map(int, input().split())
    answers.append(str(prefix[x2][y2] - prefix[x1 - 1][y2] - prefix[x2][y1 - 1] + prefix[x1 - 1][y1 - 1]))

print('\n'.join(answers))
```

---

## **✅ 잘한 점**

### **1. 문제 유형을 정확히 파악**

- 단순 반복이 아닌 **누적합으로 전처리**한 점이 핵심
- O(N^2 + M)으로 최적화된 접근

## **⚠️ 아쉬운 점**

### **1. 변수명이 조금 더 명확하면 좋음**

- prefix → prefix_sum
- row → row_values

👉 협업/유지보수 관점에서 의미 전달이 더 명확해짐

---

### **2. 핵심 공식에 대한 주석 부족**

- 2차원 누적합은 직관적이지 않기 때문에
    
    한 줄 설명이 있으면 가독성이 크게 향상됨

## **🧠 배운 점**

- 구간 합 문제는 → **누적합으로 변환**
- 2차원에서는 → **겹치는 영역 보정이 핵심**
- 반복 계산 대신 → **전처리 + O(1) 질의**
