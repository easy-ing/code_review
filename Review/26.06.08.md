# **📌 대충 만든 자판**

**문제 유형**

해시맵, 문자열 탐색, 인덱스관리

---

### 코드

```python
def solution(s):
    answer = []
    last_index = {}

    for i, char in enumerate(s):
        if char in last_index:
            answer.append(i - last_index[char])
        else:
            answer.append(-1)

        last_index[char] = i

    return answerㅍ
```

---

## **핵심 아이디어**

각 문자가 **마지막으로 등장한 위치**를 저장하면 된다.

문자열을 왼쪽부터 순회하면서

- 처음 나온 문자라면 `1`
- 이전에 나온 문자라면 `현재 인덱스 - 이전 인덱스`

를 `answer`에 넣는다.

그 후 현재 문자의 위치를 딕셔너리에 갱신한다.

---

## **시간 복잡도**

- 문자열 s를 한 번만 순회한다. 따라서 시간복잡도는 O(N)이다,

---

## **총평**

이 문제는 매번 앞쪽 문자열 전체를 다시 탐색하면 비효율적이다.

대신 **각 문자의 최근 등장 위치를 딕셔너리에 저장**하면 한 번의 순회로 해결할 수 있다.

핵심은 같은 문자가 다시 나올 때마다 위치를 최신 인덱스로 갱신하는 것이다.
