# **📌 햄버거 만들기**

**문제 유형**

스택, 배열 탐색, 패턴 매칭, 구현

---

### 코드

```python
def solution(ingredient):
    answer = 0
    stack = []

    for item in ingredient:
        stack.append(item)

        if stack[-4:] == [1, 2, 3, 1]:
            answer += 1

            for _ in range(4):
                stack.pop()

    return answer
```

---

## **핵심 아이디어**

햄버거 완성 순서는 [1, 2, 3, 1]이다.

재료가 순서대로 들어올 때마다 스택에 쌓고,

스택의 마지막 4개가 `[1, 2, 3, 1]`이면 햄버거 1개를 만든 것으로 처리하면 된다.

햄버거를 만들면 해당 재료 4개는 제거해야 하므로 `pop()`으로 빼준다.

---

## **시간 복잡도**

ingredient를 한번 순회한다.

재료의 개수를 N이라고 하면 기본 순회는 O(N)이다.

---

## **총평**

이 문제는 단순히 배열 안에서 `[1, 2, 3, 1]`을 찾는 문제가 아니다.

햄버거가 만들어지면 해당 재료가 사라지고,

그 뒤에 남은 재료끼리 다시 새로운 햄버거를 만들 수 있다.

그래서 일반 탐색보다 **스택에 쌓으면서 마지막 4개를 확인하는 방식**이 적합하다.
