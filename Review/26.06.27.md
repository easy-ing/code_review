# **📌 푸드 파이트 대회**

**문제 유형**

문자열, 구현, 반복문

---

### 코드

```python
def solution(food):
    left = ""

    for i in range(1, len(food)):
        left += str(i) * (food[i] // 2)

    return left + "0" + left[::-1]
```

---

## **핵심 아이디어**

두 선수가 **같은 음식 개수**를 먹어야 하므로, 각 음식 개수를 `2`로 나눈 몫만 사용한다.

예를 들어 음식이 3개 있으면

3 // 2 = 1 즉, 각 선수에게 1개씩 배치할 수 있다.

왼쪽 절반을 만든 뒤, 왼쪽 + 물(0) + 왼쪽을 뒤집은 것 형태로 만들면된다

---

## **시간 복잡도**

음식 종류 수를 n이라고 하면 한번 순회하므로

시간 복잡도는 O(N)이다

---

## **총평**

이 문제는 **대칭 문자열 만들기** 문제이다.

핵심은 `food[0]`은 물이므로 제외하고, `1번 음식부터` 각 음식의 절반만 왼쪽에 배치하는 것이다.

마지막에 가운데 `0`을 넣고, 왼쪽 문자열을 뒤집어서 붙이면 정답이 된다.
