# **📌 문자열 나누기**

**문제 유형**

문자열(String), 반복문(For), 조건문(If), 카운팅

---

### 코드

```python
def solution(s):
    answer = 0
    x_count = 0
    other_count = 0
    x = ''
    
    for char in s:
        if x_count == 0:
            x = char
        
        if char == x:
            x_count += 1
        else:
            other_count += 1
        
        if x_count == other_count:
            answer += 1
            x_count = 0
            other_count = 0
    
    if x_count != 0:
        answer += 1
    
    return answer
```

---

## **핵심 아이디어**

- 현재 분리할 문자열의 첫 글자를 `x`로 설정
- 문자열을 왼쪽부터 하나씩 확인
- `x`와 같은 문자의 개수는 `x_count`
- `x`와 다른 문자의 개수는 `other_count`
- 두 개수가 같아지는 순간 문자열을 하나 분리했다고 판단
- 마지막에 개수가 같아지지 않은 문자열이 남아 있다면 한 번 더 카운트

---

## **시간 복잡도**

- 문자열 `s`를 한 번만 순회함
- 문자열 길이를 `n`이라고 하면 전체 시간 복잡도는 O(n)

---

## **총평**

문자열을 한 글자씩 순회하면서 기준 문자와 다른 문자의 개수를 비교하는 방식으로 문제를 깔끔하게 해결한 코드입니다.

새로운 문자열이 시작될 때 기준 문자 `x`를 다시 설정하고, 두 카운트가 같아질 때마다 초기화하는 흐름이 문제 조건과 잘 맞습니다.
